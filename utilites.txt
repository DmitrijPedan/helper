Utilites

$ sudo apt install hardinfo      // графический диспетчер устройств 
$ sudo apt install -y htop       // монитор системных ресурсов Linux
$ sudo apt install -y mc	     // коммандер
$ sudo apt install -y make	     // создание команд
$ sudo snap install postman      // postman

Cdemu (CD/DVD images):
